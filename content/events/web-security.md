# Web Security Workshop: Understanding OWASP Top 10

Dive deep into web application security by exploring the OWASP Top 10 most critical security risks.

## Workshop Overview

This hands-on workshop will guide you through:
- SQL Injection attacks and prevention
- Cross-Site Scripting (XSS) vulnerabilities
- Authentication and session management flaws
- Security misconfigurations

## What to Bring

- Laptop with a modern browser
- Basic understanding of HTML/JavaScript helpful but not required

We'll provide access to our online lab environment for safe, legal practice!
